webMonitorMain is simple websites monitor.

To use it type in ConfigFile websites which you want to test and write content
requirements which corresponding page shall have. There is also possibility so set amount of checks by amountOfChecks
variable and delay between checks byt delayBetweenCheck variable. Result will be stored in WebMonitorLog.log file.
Result contains information like: website loading time, fulfillment of requirements, problems with opening page.

To run HTTPServer test - firs run HTTPServer.py and then run webMonitorMain.py
